## Aufgabe 1

Sie werden gebeten, für den Chocolatey-Server ein neues Paket zu
erstellen Die folgenden Daten wurden Ihnen hierfür bereitgestellt:

| Paramater          | Wert                              |
|--------------------|-----------------------------------|
| Paketname (Id)     | notepadpp862                      |
| Paketversion       | 1.0                               |
| Installationsdatei | npp. 8. 6. 2. Installer. x64. exe |
| Chocolatey-Server  | 10.0.0.2                          |

a)  ::: {.subtask}
    Befüllen Sie die Dateien „notepadpp.nuspec" und
    „chocolateyInstall.ps1", damit ein entsprechendes Paket erstellt
    werden kann. Ergänzen Sie weitere Parameter wie z. B. „title" oder
    „authors" mit sinnvollen Informationen.
    :::

<!-- -->

b)  ::: {.subtask}
    Geben Sie den Befehl an, mit dem das Paket auf dem Chocolatey-Server
    abgelegt wird.
    :::
:::
